\# ProxyApp



Приложение для управления, проверки и использования прокси‑серверов.



\## Функциональность

\- Поддержка нескольких протоколов: HTTP, SOCKS5, MTProto

\- Подключение к приложениям, поддерживающим прокси

\- Автоматическая проверка публичных прокси каждые 5 минут

\- Добавление прокси вручную

\- Поддержка авторизованных прокси

\- Локальное хранение данных через Room

\- Гибкая система фильтрации

\- Локальное хранение конфигураций через DataStore Preferences

\- Поддержка светлой, темной и системной темы



\## Источники прокси

ProxyApp может получать прокси из нескольких источников.

\### Публичные прокси

Приложение может загружать список публичных прокси из:



`https://freeproxydb.com/`

\### Пользовательские прокси

Пользователь может добавить прокси вручную.



Поддерживается:

\- IP‑адрес

\- Порт

\- Выбор протокола

\- Авторизация (username / password)

\- Секретный ключ для MTProto



После загрузки прокси автоматически проходят проверку доступности.



Интервал проверки: \*\*каждые 5 минут.\*\*



\## Фильтрация прокси

В приложении реализована система фильтров для быстрого поиска нужных прокси.



Поддерживаются фильтры:

\- Только активные прокси

\- Тип протокола

\- Диапазон скорости

\- Страна

\- Прокси, совместимые с Telegram



\## Интеграция с Android‑системой

Интеграция прокси с системными сетевыми настройками Android \*\*в разработке.\*\*



\## Технический стек

\- \*\*UI\*\*: Jetpack Compose

\- \*\*Навигация\*\*: Navigation Component

\- \*\*База данных\*\*: Room

\- \*\*Хранилище конфигураций\*\*: DataStore (Preferences)

\- \*\*DI\*\*: Hilt

\- \*\*Сетевое взаимодействие\*\*: Retrofit, OkHttp

\- \*\*JSON-сериализация / десериализация\*\*: Moshi

\- \*\*Асинхронность и потоковые данные\*\*: Coroutines, Flow



\## Архитектура и структура проекта



Проект построен по принципам \*\*Clean Architecture + MVI + Multi-Module\*\*



\### Модули



\- \*\*app\*\* - основной Android-модуль

\- \*\*data\*\* - мапперы, репозитории, DataSources, Hilt модули

\- \*\*domain\*\* - модели, интерфейсы репозиториев, UseCase

\- \*\*core-common\*\* - общие утилиты

\- \*\*core-ui\*\* - общие компоненты UI

\- \*\*feature\*\* - фичи:

&#x20; - `feature-proxy-list` - экран списка прокси и фильтрации

&#x20; - `feature-proxy-setup` - экран добавления и подключения прокси

&#x20; - `feature-settings` - экран настройки приложения

&#x20; - `feature-splash` - сплэш-экран
